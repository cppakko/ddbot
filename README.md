# DDbot

一个没什么屁用的开播提醒机器人🧰

##### ～誰でも大好き～

在go-cqhttp工作目录下⬇

``` bash
docker run -i --name bot --net=host -v $(pwd):/home/bot --restart=always bot:v1 sh boot.sh
```

boot.sh⬇

``` bash
java -jar bot.jar
```
